The basics

Webdesign

Ik heb bij webdesign HTML, CSS, javascript en bootstrap geleerd.

HTML is eigenlijk de basis van een site is. HTML zorgt ervoor dat er tekst op je site aanwezig is. Dit heb ik geleerd met behulp van college, begeleid studeren en zelfstudie in de vorm van W3Schools. Ik heb bijvoorbeeld over headers geleerd en paragraphs.

CSS kan de layout van je site veranderen, waardoor de site er als een moderne site uitziet. Door bijvoorbeeld de achtergrond kleur te veranderen of de kleur van tekstkopjes aan te passen. Het handigste is om CSS niet in je HTML file te verwerken, maar in een aparte CSS file deze code te schrijven. Hier door is deze CSS file te gebruiken op andere pagina's.

Javascript maakt je site meer interactief, dit houdt in dat de gebruiker ergens op kan klikken en dat die knop dan iets doet. De site is hierdoor niet alleen een statische site, maar dynamisch, dat is ook leuker voor een gebruiker.

Bootstrap is een framework die het makkelijker maakt om sites te maken. Je gebruikt een CSS file waar al code instaat, hierdoor kan je een class geven aan een element in je html file en krijgt dat element al een bepaalde site. Veel moderne sites gebruiken bootstrap.

Programmeren

Bij programmeren hebben ik de basis van Java gehad. Ik heb geleerd over classes, objecten, atributen methodes, constructoren en variable de echte basis van het programmeren is behandeld.
Laatst heb ik ook geleerd over het ombouwen van code. Dit houd in dat je vanplaats een grote main class heb meerdere verschillende class files gebruikt en de main class eigenlijk maar één regel bevat.

We hebben allemaal kleine applicaties gemaakt met programmeren, maar dit zijn allemaal kleine test applicaties, zoals een btw calculator. Ik ben wel bezig met een echt grote applicatie waar ik veel aan heb gewerkt samen met mede student Niels. Deze applicatie ben ik met hem aan het maken voor de DUO opdracht van programmeren en relationle databases.. To be continued.

Relationele databases

Bij relationele databases heb ik geleerd over SQL en dan vooral hoe je met behulp van code tabellen kan toevoegen, updaten en verwijderen. Ook hebben we een ERD gemaak, dit is een tekening waarbij je de database meer visualiseert voor dat je de tabellen en kolommen ook werkelijk maakt in je SQL.

Studeren op het HBO

Bij studeren op het HBO heb ik geleerd hoe je het best kan studeren op Avans Hogeschool. Ook heb ik geleerd hoe je het best kan plannen en hoe je een actieplan moet maken. Dit actieplan is vooral erg handig omdat het het plannen veel makkelijker maakt en je voor jezelf echt een duidelijk doel zet.

Beroepsoriëntatie

Bij beroepsoriëntatie heb ik geleerd om te denken over een baan voor later. Ik ben dan ook druk bezig met een beroeps onderzoek, over welke beroepen er mogelijk zijn in de informaticasector en welke het best bij mij past. Ook heb ik geleerd over interviewtechnieken dit ga ik gebruiken om iemand met een baan in de sector informatica te interviewen

Bedrijfsprocessen

Bij bedrijfsprocessen heb ik geleerd de processen binnen een bedrijf te visualiseren in een UML. Dit is een beetje te vergelijken met een ERD bij relationele databases. Alleen ziet deze visualisering er wel anders uit. Zelf vind in deze visualisering lastiger dan bij ERD, maar dit vak hebben we ook nog niet heel de periode gehad.

Leerwijze

Mijn leerwijze is tot nu toe vooral de presentaties goed bestuderen en dan veel oefenen. Ik denk zelf dat heel veel doen en herhaling de beste manier is om wat te leren. Fouten maken maakt niet uit, als je erachter komt waarom en wat de fout is leer je daarvan.

Moeilijkheden

Bij Java en Javascript had ik vooral veel moeite met loops, maar door goed onderzoek te doen op w3schools naar loops in javascript en goed alle powerpoints opnieuw te bestuderen heb ik het idee dat ik loops nu ook onder controle heb. In het algemeen heb ik ook wel moeite met verslagen schrijven of grote stukken tekst schrijven. Ik denk dat ik hier nog heel veel voor moet oefenen om dit compleet onder de knie te krijgen.

Ambities

Ik wil heel graag verder gaan met webdesign. Ik wil vooral graag meer over bootstrap leren, de verschillende classes, waardoor een site echt een mooie moderne uitstraling krijgt. Ook heb ik heel veel zin om met programmeren verder te gaan, vooral het idee van het maken van android apps spreekt me erg aan.
