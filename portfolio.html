The basics

Webdesign

Ik heb bij webdesign HTML, CSS, javascript en bootstrap geleerd.

HTML: Bij HTML heb ik geleerd dat HTML eigenlijk de basis van een site is. HTML zorgt ervoor dat er tekst op je site aanwezig is. Dit heb ik geleerd met behulp van college, begeleid studeren en zelfstudie in de vorm van W3Schools. Ik heb bijvoorbeeld over headers geleerd en paragraphs.

CSS: Bij CSS heb ik geleerd hoe je de layout van een site kan veranderen, waardoor de site er als een moderne site uitziet. Door bijvoorbeeld de achtergrond kleur te veranderen of de kleur van tekstkopjes aan te passen. Het handigste is om CSS niet in je HTML file te verwerken maar in een apart CSS file deze code te schrijven. Hier door is deze CSS file te gebruiken op andere paginas.

Javascript: Bij javascript heb ik geleerd dat javascript je site interactief maakt. Dit houdt in dat de gebruiker ergens op kan klikken en dat die knop dan iets doet. De site is hierdoor niet alleen een statiche site, maar dynamisch. Dit is ook leuker voor een gebruiker.

Bootstrap: Bootstrap is een framework die het makkelijker maakt om sites te maken. Je gebruikt een css file waar al code instaat, hierdoor kan je een class geven aan een element in je html file en krijgt dat element al een bepaalde site. Veel moderne sites gebruiken bootstrap.

Mijn manier bij webdesign is vooral oefenen geweest. Deze site is daar een voorbeeld van ik heb heel veel tijd gestoken in webdesign, ook omdat het me tot nu toe het meest aantrekt van alle vakken. Ik ben tot nu toe wel elke dag bezig geweest aan deze site en ik denk zeker dat dat werk een mooi resultaat heeft geleverd.

In het begin vond ik javascript wel erg lastig, maar door goed onderzoek te doen op w3schools en goed alle powerpoints opnieuw te bestuderen heb ik het idee dat ik Javascript nu ook onder controle heb.

Ik wil heel graag verder gaan met webdesign. Ik wil vooral graag meer over bootstrap leren, de verschillende classes, waardoor een site echt een mooie moderne uitstraling krijgt.
